Stats is a web application to host anonymous analytics that report the usage of Tuist. It is accessible on https://stats.tuist.io.
If you are wondering why we are tracking Tuist's usage, you can read the [FAQ](https://stats.tuist.io/#faq).

It's built in Rails and hosted on Heroku.

## Setup

1. Git clone the repository: `git clone git@github.com:tuist/stats.git`.
2. Install dependencies with `bundle install`.
3. Run the app with `bundle exec rails start`.
